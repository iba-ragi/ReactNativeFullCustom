## Setup
1. Install modules
```shell
$ yarn install && cd ios && pod install && cd ..  
```
2. Build
```shell
yarn dev:ios // yarn dev:android
```

## Debugging
- Boot `Teact Native Debugger`
```shell
$ yarn debugger
```

- Watch/Auto-Compile by `Typescript`
```shell
$ yarn tsc
```

## Troubleshooting
```shell
$ yarn clean
```
